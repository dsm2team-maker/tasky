# 🎨 Tasky

Plateforme de mise en relation entre clients et artisans locaux.

## 🚀 Technologies

- Next.js 14
- TypeScript
- Tailwind CSS
- Design System custom
- Zustand (state management)

## 📦 Installation

```bash
npm install
npm run dev
```

## 🎨 Design System

Toutes les couleurs, typographies et composants sont centralisés dans `/src/config`
